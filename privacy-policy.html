EnableFlow Guide

Privacy Policy

Effective Date: April 7, 2026

Introduction

EnableFlow Guide ("the Extension") is a Chrome extension that provides interactive tooltip-based training guides for Oracle EPM applications. This privacy policy explains what data the Extension collects, how it is used, and how it is stored.

In short: The Extension stores all journey data locally on your device. The only external communication is license validation with our server.

Data We Collect

Data Purpose Stored Where
Journey JSON files To display step-by-step training guides Locally (chrome.storage.local)
Journey progress (current step) To resume where you left off Locally (chrome.storage.local)
Tooltip position preference To remember your display preference Locally (chrome.storage.local)
Email address License validation Locally & sent to license server
License key License validation Locally & sent to license server
System UUID (randomly generated) Machine identification for license activation Locally & sent to license server

How Data Is Used

  • Journey data and preferences are used solely to power the tooltip guide functionality and are stored entirely on your device.
  • License information (email, license key, and system UUID) is sent to our license validation server to verify your license status. This communication occurs when you activate a license and periodically (every 24 hours) to re-validate.

Data Sharing

We do not sell, trade, or share your personal information with third parties. License validation data is sent only to our license server for the purpose of verifying your license.

Data Storage and Security

  • All journey data and user preferences are stored locally on your device using Chrome's built-in storage API.
  • License validation requests are transmitted over HTTPS.
  • No data is collected for analytics, advertising, or tracking purposes.

Permissions

The Extension requires the following Chrome permissions:

  • activeTab — To inject the tooltip guide into the currently active Oracle EPM page.
  • storage — To save journey data, preferences, and license information locally.
  • Host permissions (*.oraclecloud.com, *.ocs.oraclecloud.com, *.oraclecloudapps.com) — To run the content script on Oracle EPM pages and communicate with the license validation server.

Your Choices

  • You can clear all stored data by removing and reinstalling the Extension.
  • You can change your license by using the "Change License" option in the Extension popup, which clears your stored license data.
  • You can uninstall the Extension at any time, which removes all locally stored data.

Children's Privacy

The Extension is not directed at children under the age of 13. We do not knowingly collect personal information from children.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated effective date.
